RasDash

A Raspberry Pi dashboard built on NodeJS.

Easy to Read

With simple widgets, and an easy on the eyes style, RasDash puts readibility first.

Simple Logs

When debugging manually, simple logs can be a great help. That's why RasDash uses Ben's logger library!

Service Mode

With the help of Ernie's service manager, you can install RasDash as a service on your Raspberry Pi.

Multiple Layers

Being a multi-layered application, RasDash makes expansion easy. Wanna change the API? look in the API file! Need a new console command? Add it to the CLI!

Portable

RasDash is designed to be easily ported for use on other systems. In fact, we have a live demo running on a Heroku server!
